# College ROI Visualization Dashboard

Interactive data visualization dashboard analyzing U.S. college return on investment (ROI)
using College Scorecard data.

## Motivation
Evaluating the financial outcomes of higher education is increasingly complex due to rising
tuition costs and student debt. This project presents interactive visualizations to support
informed decision-making.

Interactive visualizations help users better understand complex educational finance data.
